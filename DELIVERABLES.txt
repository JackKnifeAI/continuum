USAGE EXAMPLE
-------------
from continuum.extraction import AutoMemoryHook
from pathlib import Path

# Create hook with voting enabled
hook = AutoMemoryHook(
    db_path=Path("memory.db"),
    use_voting=True,  # Enable ensemble voting
    llm_extractor=my_llm_function  # Optional
)

# Process messages
stats = hook.save_message("user", "Tell me about neural networks")

# Get extraction logs
logs = hook.get_extraction_logs(limit=5)
for log in logs:
    print(f"Extractors: {log['extractors_used']}")
    print(f"Concepts: {log['total_concepts_found']}")
    for detail in log['concept_details']:
        print(f"  - {detail['concept']}: {detail['confidence']:.2f}")

DESIGN RATIONALE
----------------
Single extractors have limitations:
  - Regex: Fast but misses semantic variations
  - Semantic: Catches synonyms but noisy
  - LLM: High quality but slow and may hallucinate

Solution: Ensemble voting combines all extractors
  - Better accuracy through consensus
  - Configurable precision/recall tradeoff
  - Confidence scores for transparency
  - Metrics for monitoring

Default configuration (WEIGHTED):
  - regex=0.3 (fast, baseline)
  - semantic=0.5 (catches synonyms)
  - llm=0.8 (highest quality)
  - threshold=0.4 (balanced)
