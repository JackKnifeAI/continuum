## Name
CONTINUUM - AI Memory

## Short Description (80 chars)
Your AI never forgets. Capture, recall, and connect knowledge across the web.

## Long Description

CONTINUUM transforms your browser into an AI-powered memory system. Never lose track of important information again.

### Key Features

**Effortless Capture**
- Highlight and save text with one click
- Automatically extract article content, code snippets, and videos
- Context menu integration for quick saving
- Offline queue syncs when you're back online

**Smart Recall**
- Search your entire knowledge graph from any page
- See captured content highlighted on pages you visit
- Get instant context for what you're reading
- Related memories surface automatically

**Powerful Integration**
- Sidebar for browsing all memories
- Concept graph explorer
- Keyboard shortcuts for power users
- Works with your self-hosted CONTINUUM instance

**Privacy First**
- All data goes to YOUR CONTINUUM server
- No tracking or third-party services
- Open source and auditable
- You control your data

### How It Works

1. Install the extension
2. Configure your CONTINUUM API endpoint
3. Start capturing content as you browse
4. Search and recall anytime, anywhere

### Who It's For

- Researchers managing sources and citations
- Developers saving code snippets and documentation
- Students building study materials
- Anyone who wants to remember what they read

### Technology

Built with modern web technologies:
- React for UI
- TypeScript for reliability
- Manifest V3 for security
- Works offline

### Requirements

Requires a CONTINUUM instance (self-hosted or cloud). Get started at continuum.ai
